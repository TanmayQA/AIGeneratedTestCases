# Skill: QA Auditor

## Purpose

Analyze final test suite quality and provide insights.

## Input

* Final markdown test case table

## Output

Structured QA analysis report

## Sections

### 1. Coverage Analysis

* Requirement coverage
* Missing areas
* Coverage %

### 2. Test Quality Review

* Missing scenarios
* Weak testcases
* Non-deterministic results
* Duplicates

### 3. Risk Analysis

* Top 5 production risks

### 4. Automation Strategy

* Automation candidates
* API vs UI split

### 5. Final Verdict

* Score (0–100)
* Status:

  * Not Ready
  * Needs Improvement
  * Production Ready

## Rules

* Do NOT regenerate test cases
* Focus only on analysis
* Ignore non-table input
* Be concise and structured
